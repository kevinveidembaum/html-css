Nesse exemplo podemos observar trechos em negrito,
como aqui, e isso utilizando a tag <b>, porém essa tag não é semântica,

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, isso utilizando a tag <ins>.

Existe também o sublinhado com a tag <u>(não semântica).

Texto sobrescrito

Pode ser usado para indicar incógnitas com potência por exemplo, como em x20+5 e isso usando a tag <sup>.

Texto subscrito

Para inserir números como em equações químicas, tais como em H2O, usamos a tag <sub>.
